# OIBSIP

This repository contains tasks completed during the Oasis Infobyte Cyber Security Internship (OIBSIP).

## Completed Tasks

### Task 1 - Basic Network Scanning with Nmap

* Installed and configured Nmap.
* Performed network scanning on a local network device.
* Identified open ports and running services.
* Analyzed the significance of discovered ports and services.
* Documented scan results and findings.

### Task 2 - Basic Firewall Configuration with UFW

* Installed UFW (Uncomplicated Firewall).
* Allowed SSH traffic (Port 22).
* Denied HTTP traffic (Port 80).
* Enabled and configured firewall rules.
* Verified active firewall configuration.

### Task 8 - Capture Network Traffic with Wireshark

- Installed Wireshark and Npcap.
- Captured network traffic on a Wi-Fi interface.
- Filtered and analyzed HTTP packets.
- Examined HTTP requests and responses.
- Saved packet capture for analysis.

## Tools Used

* Nmap
* UFW
* Windows Command Prompt
* Kali Linux
* VirtualBox
* Wireshark
